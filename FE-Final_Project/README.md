# NewsExplorer - My First React Project! 📰

**Live Demo:** [https://sam-morris01.github.io/FE-Final_Project/](https://sam-morris01.github.io/FE-Final_Project/)

Welcome to my first React project! This is a news website where you can search for articles, save your favorites, and manage your account.

## What This Project Does 🌟

**NewsExplorer** is a news website that lets you:
- 🔍 **Search for news articles** - Find articles about topics you're interested in
- 💾 **Save articles** - Keep track of articles you want to read later
- 👤 **Create an account** - Sign up and log in to save your articles
- 📱 **Responsive design** - Works on your phone, tablet, and computer

## How to Run This Project 🚀

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Step 1: Download the Project
```bash
git clone [your-repository-url]
cd FE-Final_Project
```

### Step 2: Install Dependencies
```bash
npm install
```
(This downloads all the tools and libraries the project needs)

### Step 3: Start the Development Server
```bash
npm run dev
```
(This starts the website on your computer - usually at http://localhost:5173)

### Step 4: Open in Your Browser
The website should automatically open in your browser. If not, go to http://localhost:5173

## How to Use the Website 🎯

1. **Home Page**: You'll see a search bar where you can look for news articles
2. **Search**: Type in a topic (like "technology" or "sports") and press Enter
3. **Read Articles**: Click on any article to read more about it
4. **Save Articles**: Click the bookmark icon to save articles you like
5. **Sign Up/Login**: Create an account to save your articles
6. **Saved Articles**: Click "Saved articles" to see all your saved news
